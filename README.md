# Midnight Focus 🌙

Midnight Focus is a productivity web application designed to help users stay focused and manage their time effectively.

## Features

* Focus Timer
* Short & Long Breaks
* Clean Dark/light Theme
* Responsive Design
* Productivity-Oriented Interface

## Status

🚧 Work in Progress

This project is currently under development and new features are being added regularly.

## Technologies Used

* HTML
* CSS
* JavaScript


## Goal

The goal of Midnight Focus is to provide a simple and distraction-free environment for studying, working, and maintaining focus.
